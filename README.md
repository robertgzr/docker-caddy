* x86_64

* github.com/techknowlogick/caddy-s3browser
* github.com/hacdias/caddy-minify
* github.com/abiosoft/caddy-git
* github.com/zikes/gopkg
* github.com/captncraig/cors
* github.com/nicolasazrak/caddy-cache
